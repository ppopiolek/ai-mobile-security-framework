Overview

This site presents a comprehensive framework for documenting and securing AI/ML models in next-generation mobile networks (5G/6G). The focus is on network-oriented Model Cards that capture both technical model details and their operational context within mobile network infrastructure.

Why Network-Oriented Model Cards?

Traditional Model Cards focus primarily on the model itself. In mobile networks, AI/ML models have significant impact on network operations, making it essential to document not just the model, but also its integration context, data sources, and operational dependencies.

Network-Oriented Model Cards Framework

Model Cards document an AI/ML model together with its network context: placement (e.g., NWDAF, MDAF, RIC), data sources and clients (NFs), inputs/outputs, validation methods, and operational environment.

How to Create Model Cards

Model Cards should be created by the original developers of the AI/ML system to ensure accuracy and completeness. The framework provides a standardized template that can be adapted to specific use cases in mobile networks.

Security Framework Overview

The framework emphasizes the operational context and network integration of AI models, moving beyond isolated model analysis.
